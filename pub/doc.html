fr.js Object

The fr library base object can be instantiated as

const report = new fr();

All methods should be called on this object as

report.method(...);

Methods

Method Signature

Arguments

Return Values

 

statementGenerator(data, 
id, sub, budget, edit);

See the examples page for format of the data. id argument sets table id. subtotal is a boolean to create subtotals for each data category. edit is boolean to set each value in an input form with edit and save buttons. budget must match the data object. See the examples page for precise format. sub, budget and edit are optional.

This method returns a financial report table based on user input.

 

total(select, label, 
operations, budget);

select is the id selector of the table. The label is the name given to the total row. operations is an object with either add or sub operations. See examples for format. budget is a boolean. Select true to calculate total on budgeted data. budget is an optional argument.

This method calculates totals on a report table based on the label as the title of the row.

 

editTotal(select, label, 
operations, budget);

editTotal takes the same arguments as total above. It can be used as an onchange handler to recalculate totals after changing a table value.

This method recalculates totals on a report table based with the label as the title of the row. 

draganddrop(select);

select is the id of the table.

This method makes the selected table rows draggable and droppable so the user can reorder table rows. 

formatValues();

No arguments are taken for this method. This method is best used as an onload or onchange handler to format the dollar values of each row.

This method formats the dollar values of each row and does not return any value.

 

formatDifferences();

No arguments are taken for this method. This method is best used as an onload or onchange handler to format valeus

This method formats the percentage difference between values for each row and does not return any value.

 

checkDifferences(id, threshold, 
success, failure);

id is the id selector for the table. Threshold is a integer value. success and failure are css colors.

This method highlights percentage differences greater than the threshold as sucess and lower than the negative threshold as failure. No value is returned.

 

addLink(label);

label is the name of a row within a table.

This method hyperlinks rows of tables that share the same label to show to compare their values.

 

createManualSlideShow(root, 
selector, slides);

root is the selector that the user attaches a slideshow to. selector is the class name to identify and toggle between slides. slides is an array of tables (which can be generated using statementGenerator).

This method creates a manual slideshow of tables which can be toggled using the next and previous buttons.

 

createAutomaticSlideShow(
root, selector, slides, 
speed);

The arguments for this method are the same as the manual slide show above. The additional speed component is the number of milliseconds before the slideshow transitions to the next slide.

This method creates an automatic slideshow of tables that transition between slides at the specified speed.

 

rateCalculator(rate, label, 
table, message);

Rate is a float greater than or equal to 0. label is the name of a row within a table. table is a table (generated by statementGenerator) and message is a string of text.

This method adds a calculated row which is calculated based on the rate times the label row value. This will be displayed in the table with the message.

 

addSubComponent(data, label, 
cat, table);

data is an object that maps to a row in a table. See examples page for the precise format. label is the name of a row that the data maps to. cat is the category of the label. table is a table (which is generated by statementGenerator.

This method adds subrows to a row label in a table.
